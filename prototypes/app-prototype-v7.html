总览 · OVERVIEW

15 屏全景 · 大理慢递设计版

「明信片美学」覆盖主流程、辅助页、争议申诉。admin 屏已剥离至独立 web 后台（不进小程序代码）。

屏幕清单 · SCREENS
  • 主流程 8 屏　信箱信息流 · 地图 · 寻宠详情 · 招领详情 · 发布表单 · 联系建议 · 归还自检 · 归还成功
  • 用户辅助 5 屏　感谢信编辑 · 消息列表 · 消息对话 · 个人中心 · 善行值流水
  • 管理辅助 2 屏　我的发布 · 争议申诉
  • 已剥离　后台审核 → 见独立文件 admin-web-prototype.html
合规改造 · COMPLIANCE
  • P0admin 屏剥离到独立 web 后台 — 小程序内禁止内嵌管理后台
  • P0联系方式仅站内消息 / 虚拟号中转 — 禁止明文手机号
  • P0发布须知明示禁止悬赏金 / 感谢金 / 微信号引导
  • P0UGC 三审接入点publish + chat + thanks + dispute 必接 msgSecCheck / imgSecCheck
  • P1个人中心用户协议 / 隐私政策 / 客服 / 注销 + ICP 备案
  • P1位置精度模糊至 500m 圆圈 — 数据库存 geohash 加密
  • P1善行值顶部明示不可兑换现金 / 商品
  • P1启动时隐私协议确认 modal · 信息收集明示
设计语言 · DESIGN
  • 主色 · 板蓝根靛青　大理白族扎染的灵魂色，沉静、值得托付
  • 强调 · 朱砂红印章　寻找 / 紧急 / 印戳状态
  • 荣誉 · 银杏黄　善行值 / 勋章 / 等级
  • 字体　Noto Serif SC（衬线标题）· Noto Sans SC（正文）· Cormorant Garamond（数字）
  • 纸纹底　SVG 噪声叠加 multiply 混合 — 营造明信片质感
