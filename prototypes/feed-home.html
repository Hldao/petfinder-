📐 这一版的几个设计决策（请你评判）

  • 底部 5 个 tab 改成"首页 / 地图 / + / 消息 / 我的"，把"发布"从底栏拎出来变成中间凸起的橙色悬浮按钮（FAB）—— 这是寻宠平台最关键的动作，要让它视觉上无法忽视
  • 新增 "地图" tab：大理范围小、地点高度地标化（古城北门、人民路、洱海月），地图视图比纯列表更直观
  • 卡片左上角的 状态徽章 用三色区分：🔴 寻找 / 🟢 招领 / 🟡 临时照料中
  • 卡片底栏的"我有线索"按钮：对应你 V2.0 里的线索提交流程，作为最显眼的二级动作
  • 距离用橙色突出（0.8km 这种）—— 大理用户能瞬间判断是否离自己近
  • 暂时不展示失主头像 / 善行值徽章 / 转发数等"装饰性"信息，把屏幕给真正影响判断的字段
  • 顶部"大理 ▾"是城市切换入口（M0 只有大理，但保留切换控件给后续扩展）
