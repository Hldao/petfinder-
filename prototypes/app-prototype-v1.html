🎨 设计语言

  • 主色：暖橙 #FF6B35 + 米白 #FAFAF7 —— 暖、安全、能在朋友圈截图传播不出戏
  • 状态色：寻宠红 / 招领绿 / 照料黄，全程统一，扫一眼就能识别
  • 字体：PingFang SC + SF Pro Display 后备，11–22px 渐进字号
  • 圆角：4 层（8/12/16/20）—— 卡片 16、按钮 12、徽章 6
  • 阴影：3 层（标识层级 / 卡片浮起 / 模态强调），不滥用

🆕 详情页关键设计

  • "私有特征已设置" 黄色提示卡：把 V2.0 的安全策略从文字提示变成可见 UI，发现者一眼知道该走哪条流程
  • 线索列表 "失主已标记有用" 徽章：让用户看到"自己的努力被认可"——社区氛围的关键一笔
  • 招领详情嵌入"同片区寻宠 · 自动匹配"：你最在意的 C 问题（招领→丢失反向入口）的具体形态，发现者能直接看到附近的寻宠贴并跳过去核对
  • 底栏行动按钮二选一：减少决策负担——寻宠页用「留言 / 我捡到了」；招领页用「联系 / 这是我家的猫」

🆕 个人中心关键设计

  • 善行值橙色 Hero 区 + 玻璃拟态卡：让"42"这个数字成为视觉焦点，承接你 V2.0 里"荣誉记录"的精神
  • 子项分四类：归还 / 线索 / 照料 / 感谢 —— 比单一总分更立体，让用户知道贡献的是哪类
  • "我可以临时照料" 常驻开关：让"愿意收留"的人挂在线、被需要时被推送 —— V2.0 没明说但很关键
  • 守护者徽章紧贴用户名：30 善行值=守护者 的视觉荣誉
