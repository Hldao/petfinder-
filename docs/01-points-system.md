# 01 — 善行值激励体系

> **⚠ MVP 阶段不启用** — 详见 `docs/03-mvp-scope.md` § 「关于善行值的取舍」。
> 冷启动期（用户 < 1000）积分系统是"空跑的"，MVP 用「行为统计」替代「积分数值」。
> 本文档作为 **V2 启用时的设计参考** 保留。
>
> **⚠ 「临时照料 +5」条目已随功能整体砍掉** — 详见 `docs/03-mvp-scope.md` § 「关于临时照料的取舍」。
> V2 若以「转交救助站」形式重启，激励对象将是**救助站交接成功**而非私人代管行为，规则需重新设计。
>
> ---
>
> 设计原则：按"**行为成本 + 网络贡献**"双维度定价，不按"是否结果性"。
> 让线索从"低价值参与"升级为"可期待回报的核心行为"。

## 获得规则

| 行为 | 善行值 | 限制 |
|------|--------|------|
| 提供有效线索（失主标记"有用"） | **+2** | 每条线索最多 1 次；同一用户对同一寻宠事件最多 2 条线索得分 |
| 关键线索（失主确认"促成找回"） | **+3 追加** | 失主在确认归还时勾选最多 3 条关键线索；触发追加奖励 |
| ~~临时照料（发布招领时勾选可照料）~~ | ~~+5~~ | ❌ **已随功能砍掉**（V2 救助站方案落地后重新设计） |
| 成功归还 | **+4** | 双方确认归还 + 24h 冷却期后结算 |
| 感谢信被点名 | **+1** | 每封感谢信最多点名 3 人；每人每日最多被感谢 2 次 |
| 邀请新用户注册 | **+1** | 每月最多通过邀请获得 5 点；被邀请人需绑定手机且首次访问 ≥ 30s |
| 成功举报虚假信息 | **+1** | 同一信息给首位举报者；每人每日最多 3 次 |

### 单事件总上限 25 点

一个完整找回案件（含线索 + 关键标记 + 归还 + 感谢信）的善行值合计上限。

## 关键改动 vs 原方案

| 行为 | 原方案 | 新方案 | 改动逻辑 |
|------|--------|--------|---------|
| 提供有效线索 | +1 | **+2** | 线索是网络密度的根本来源，基础奖励要够 |
| 关键线索（**新增**） | — | **+3 追加** | 鼓励高质量线索，不是刷数量 |
| ~~临时照料~~ | ~~+2~~ | ❌ 砍 | 风险面过广（疾病传播 / 二次走失 / 法律灰色），整条功能线在 MVP 移除 |
| 成功归还 | +6 | **+4** | 不再相对临时照料定价，按"行为成本"独立评估 |
| 单事件上限 | 15 | **25** | 完整链路就该多奖 |

## "线索质量"反馈机制

失主标记线索改为**三档**（不是二元）：

- ✅ **有用** — 提供了有价值信息（+2）
- ⭐ **关键促成** — 直接促成找回（+2 + 追加 +3 = 5 点）
- ❌ **错误** — 信息不实或无关（不得分；恶意虚假线索可被举报扣分）

UI 设计：失主在归还确认页可指认最多 3 条关键线索（一次性操作，不可撤销）。

## 善行值在 MVP 的"出口"

⚠️ MVP 不做兑换/捐赠，但**必须有荣誉激励**，否则数字攒不下去。

### 月度仪式感设计

1. **月度善意榜** — 每月 1 日公开当月 Top 10，含头像 + 善行值 + 主要贡献（线索 / 归还 / 照料）
2. **月度徽章** — 月度榜入选自动获得徽章，永久挂在个人主页
3. **守护者徽章** — 累计善行值 ≥ 50 自动获得，徽章永久
4. **运营公众号 / 朋友圈月报** — 开发者亲自整理"X 月善意月报"，发布在合作群 + 朋友圈，带成功案例和数据

### V2 出口（不在 MVP 范围）

- 善行值兑换捐赠（合作救助组织时）
- 善行值兑换专业寻宠师付费咨询券
- 守护者社区线下聚会优先权

## 防作弊机制

| 机制 | 规则 |
|------|------|
| **单事件总上限** | 同一寻宠事件累计 ≤ 25 点 |
| **同一用户对单事件累计上限** | 线索类最多 2 条得分（4 点）+ 关键线索追加最多一次（3 点） |
| **时间冷却** | 同一用户对同一宠物每 24h 只能新增 1 次善行值 |
| **邀请月上限** | 每人每月最多 5 点 |
| **每日感谢上限** | 每用户每日最多被感谢 2 次 |
| **24h 归还冷却** | 双方确认归还后 24 小时才结算，期间任一方可撤回 |
| **每周抽检** | 随机抽 10% 的"有用标记"和"归还确认"，志愿者联系用户核实 |
| **作弊惩罚** | 首次：双方善行值清零 + 封号 7 天；二次：永久封号 + 历史总值归零 |
| **小号关联** | 同设备/手机号注册的多账号行为联动监控 |
